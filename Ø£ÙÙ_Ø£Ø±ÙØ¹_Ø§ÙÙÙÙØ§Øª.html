أين أرفع الملفات؟ GitHub

دليل مصوّر: مكان رفع ملفات ROOK Sport بالضبط من هاتفك 📱

1 افتح مستودعك على GitHub

بعد إنشاء المستودع rook-sport، افتح صفحته. ستجد شريط التبويبات في الأعلى:

🔒github.com/اسمك/rook-sport
CodeIssuesPull requestsActionsSettings

المستودع فارغ الآن — هنا سترفع الملفات ⬇️

2 اضغط زر «Add file»

في صفحة Code، ابحث عن زر Add file (عادةً أعلى يمين قائمة الملفات) واضغطه:

🔒github.com/اسمك/rook-sport
＋ Add file ▾ 👆 <> Code ▾

ستظهر قائمة منسدلة، اختر منها:

📝 Create new file
📤 Upload files ← اختر هذا

3 ارفع الملفات هنا

ستفتح صفحة الرفع. اسحب الملفات أو اضغط choose your files لاختيارها من جوالك:

🔒.../rook-sport/upload/main
📁 اسحب الملفات هنا
أو اضغط «choose your files»
👆

اختر كل الملفات والمجلدات داخل مجلد ZonaSportApp

▼

ثم انزل لأسفل واكتب أي وصف، واضغط الزر الأخضر:

Commit changes 👆

⚠️ أهم نقطة: المجلد المخفي .github

مجلد .github (الذي يبدأ بنقطة) يحتوي تعليمات بناء الـ APK. بعض تطبيقات الملفات تخفيه. بدونه لن يعمل البناء!

الحل الأسهل: ارفع الملفات على عدة دفعات، وتأكد أن تطبيق إدارة الملفات لديك مفعّل فيه خيار «إظهار الملفات المخفية».

💡 بديل ذكي إن لم يظهر: أنشئ الملف يدوياً —
اضغط Add file → Create new file، واكتب في خانة الاسم بالضبط:
.github/workflows/build-apk.yml
ثم الصق محتوى الملف (موجود في المشروع)، واحفظ. GitHub سينشئ المجلدات تلقائياً!

4 شغّل البناء

بعد رفع كل الملفات، اذهب لتبويب Actions في الأعلى:

🔒github.com/اسمك/rook-sport
CodeActions ← 👆Settings
▶️ بناء تطبيق ROOK Sport (APK)

اضغط عليه ← ثم زر Run workflow الرمادي

بعد ~5 دقائق تظهر علامة ✅ خضراء. اضغط عليها → انزل لأسفل → قسم Artifacts → نزّل ROOK-Sport-APK. فُكّ الضغط ← ثبّت app-debug.apk. تم! 🎉
