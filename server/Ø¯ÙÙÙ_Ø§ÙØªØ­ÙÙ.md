# 🎛️ دليل التحكم في ROOK Sport (بدون تحديث التطبيق)

هذا الدليل يشرح كيف **تضيف مباريات** و**تحدّث روابط البث m3u8** في أي وقت، وتظهر فوراً لكل المستخدمين **بدون أي تحديث للتطبيق**.

---

## 🧠 الفكرة باختصار
- التطبيق عند كل فتح يقرأ ملفاً واحداً اسمه `config.json` من الإنترنت.
- أنت تتحكم بهذا الملف. تعدّله → تحفظه → يظهر التغيير في التطبيق فوراً.
- لا داعي لإعادة رفع التطبيق على المتجر إطلاقاً.

```
[ هاتف المستخدم ] ←—— يقرأ ——→ [ config.json على الإنترنت ] ←—— تعدّله أنت
```

---

## 🚀 خطوة 1: استضافة الملف مجاناً (GitHub)
1. أنشئ حساباً على https://github.com (مجاني).
2. اضغط **New repository** → سمِّه مثلاً `rook-config` → اجعله **Public** → Create.
3. اضغط **Add file → Upload files** وارفع ملف `config.json` (الموجود في هذا المجلد).
4. افتح الملف داخل GitHub ثم اضغط زر **Raw**.
5. انسخ الرابط من شريط العنوان، سيكون بهذا الشكل:
   ```
   https://raw.githubusercontent.com/اسمك/rook-config/main/config.json
   ```

## 🔗 خطوة 2: ربط التطبيق بالرابط (مرة واحدة فقط)
افتح الملف:
```
app/src/main/java/com/rook/sport/data/RemoteConfigUrl.kt
```
وضع الرابط مكان `URL`:
```kotlin
object RemoteConfigUrl {
    const val URL = "https://raw.githubusercontent.com/اسمك/rook-config/main/config.json"
}
```
ابنِ التطبيق مرة واحدة فقط. بعدها كل التحكم يصبح من الملف.

## ➕ كيف أضيف مباراة؟
افتح `config.json` على GitHub → اضغط ✏️ (تعديل) → أضف كائناً جديداً داخل `"matches"`:

```json
{
  "id": "match3",
  "league": "كأس العالم",
  "league_short": "المونديال",
  "home_name": "الأرجنتين",
  "home_abbr": "AR",
  "home_color": "#75AADB",
  "away_name": "فرنسا",
  "away_abbr": "FR",
  "away_color": "#002654",
  "is_live": true,
  "minute": "''45",
  "kickoff": "22:00",
  "day_label": "اليوم",
  "servers": [
    {
      "id": "s1",
      "name": "السيرفر الأول",
      "language": "عربي",
      "quality": "1080p",
      "url": "https://example.com/match3.m3u8"
    }
  ]
}
```
احفظ (Commit changes) → افتح التطبيق → ستظهر المباراة فوراً. ✅

## 🔄 كيف أحدّث رابط البث؟
نفس الطريقة: غيّر قيمة `"url"` فقط داخل `servers`، واحفظ. التطبيق سيشغّل الرابط الجديد مباشرة. ✅

## 🗑️ كيف أحذف مباراة؟
احذف كائنها من القائمة `"matches"` واحفظ.

## 📺 كيف أضيف/أحدّث قناة؟
نفس الطريقة لكن داخل قائمة `"channels"`.

## 🧾 شرح كل الحقول

### مباراة (match)
| الحقل | الوصف | مثال |
|------|-------|------|
| `id` | معرّف فريد (لا تكرره) | `"match3"` |
| `league` | اسم البطولة | `"دوري أبطال أوروبا"` |
| `league_short` | اسم مختصر | `"UCL"` |
| `home_name` / `away_name` | اسم الفريق | `"ريال مدريد"` |
| `home_abbr` / `away_abbr` | اختصار حرفين | `"RM"` |
| `home_color` / `away_color` | لون الفريق HEX | `"#C8102E"` |
| `home_score` / `away_score` | النتيجة (اختياري) | `2` |
| `is_live` | هل المباراة الآن مباشرة؟ | `true` / `false` |
| `minute` | الدقيقة الحالية | `"67'"` |
| `kickoff` | وقت البداية | `"21:00"` |
| `day_label` | اليوم/غداً | `"اليوم"` |
| `servers` | قائمة روابط البث | (انظر بالأسفل) |

### سيرفر (server)
| الحقل | الوصف |
|------|-------|
| `id` | معرّف السيرفر (s1, s2...) |
| `name` | الاسم المعروض |
| `language` | اللغة/المعلق |
| `quality` | الجودة (1080p, 720p...) |
| `url` | **رابط m3u8 / HLS** |

### إعدادات عامة
| الحقل | الوصف |
|------|-------|
| `notice` | شريط إعلان يظهر أعلى الرئيسية (اتركه `""` لإخفائه) |
| `force_update` | لإجبار التحديث لاحقاً |

## ⚠️ نصائح مهمة
1. **تحقق من صحة JSON** قبل الحفظ عبر https://jsonlint.com حتى لا يتعطل التطبيق.
2. روابط `.m3u8` يجب أن تكون **قانونية** (قنوات مرخّصة/مفتوحة). تجنّب القنوات المقرصنة.
3. إن فشل تحميل الملف، يعرض التطبيق بيانات تجريبية تلقائياً (لا يتعطل).
4. استخدم **المحرر المرئي** `محرر-الاعدادات.html` في هذا المجلد لتوليد JSON بسهولة دون كتابة يدوية.
