# 🚀 ابدأ من هنا — دليل ROOK Sport الشامل

هذا الدليل يجمع **كل الخطوات بالترتيب** من الصفر حتى تثبيت التطبيقين على جوالك — كل ذلك **من الهاتف فقط** عبر GitHub Actions.

---

## 📋 ما ستحصل عليه في النهاية
- 📱 **ROOK Sport** (تطبيق المستخدم) — تنشره للناس.
- 🔐 **ROOK Admin** (تطبيق الأدمن) — تتحكم به في المباريات والروابط.
- 🔥 قاعدة Firebase تربطهما: تعدّل من الأدمن → يظهر فوراً للمستخدم.

---

## 🗺️ الخريطة الكاملة (5 مراحل)

```
[1] إعداد Firebase  →  [2] رفع المشروع على GitHub  →  [3] بناء APK
        ↓                        ↓                          ↓
   google-services.json    تبويب Actions            تنزيل من Artifacts
                                                          ↓
                                              [4] تثبيت  →  [5] استخدام الأدمن
```

---

## ✅ المرحلة 1: إعداد Firebase (مرة واحدة)
> الملف التفصيلي: **`دليل_Firebase.md`**

باختصار:
1. أنشئ مشروعاً على https://console.firebase.google.com
2. سجّل تطبيقين في **نفس المشروع**:
   - تطبيق المستخدم: `com.rook.sport`
   - تطبيق الأدمن: `com.rook.sport.admin`
3. نزّل ملف `google-services.json` لكل واحد وضعه في:
   - `app/google-services.json`
   - `admin/google-services.json`
4. فعّل **Firestore Database** (Start in test mode).
5. غيّر كلمة مرور الأدمن في `admin/.../AdminAuth.kt`.

## ✅ المرحلة 2: رفع المشروع على GitHub
> الملفات التفصيلية: **`أين_أرفع_الملفات.html`** و **`حل_رفع_المجلدات.html`**

1. أنشئ حساباً على https://github.com
2. أنشئ مستودعاً (Repository) جديداً.
3. ارفع ملفات المشروع. **أسهل طريقة:** فعّل **«موقع كمبيوتر / Desktop site»** في متصفح Chrome، ثم Add file → Upload files → اختر المجلد.

> 💡 يمكنك رفع النسخة الكاملة `ROOK-Sport-كامل.zip` (بعد فكّها) في مستودع واحد،
> أو فصل كل تطبيق في مستودع خاص باستخدام `ROOK-User.zip` و`ROOK-Admin.zip`.

## ✅ المرحلة 3: بناء الـ APK
> الملف التفصيلي: **`كيف_تبني_APK_من_الهاتف.md`**

1. في مستودعك، افتح تبويب **Actions**.
2. اختر الـ workflow → اضغط **Run workflow**.
3. انتظر ~5 دقائق حتى تظهر ✅.
4. نزّل الـ APK من قسم **Artifacts**:
   - `ROOK-Sport-USER-APK`
   - `ROOK-Admin-APK`

## ✅ المرحلة 4: التثبيت
1. فُكّ ملف الـ APK المنزّل (تطبيق ZArchiver).
2. ثبّت `app-debug.apk` (المستخدم) و`admin-debug.apk` (الأدمن).
3. فعّل «التثبيت من مصادر غير معروفة» عند الطلب.

## ✅ المرحلة 5: الاستخدام
1. افتح **ROOK Admin** → أدخل كلمة المرور.
2. اضغط ➕ لإضافة مباراة → املأ البيانات وروابط m3u8 → **حفظ**.
3. افتح **ROOK Sport** (المستخدم) → ستجد المباراة ظهرت فوراً! 🎉

---

## 🔄 الاستخدام اليومي (بعد الإعداد)
- لإضافة/تعديل مباراة أو رابط: فقط من **تطبيق الأدمن** على جوالك — لا تحتاج GitHub ولا بناء جديد.
- لتعديل **تصميم/ميزات** التطبيق نفسه: عدّل الكود على GitHub ثم Run workflow.

## ❓ لماذا GitHub Actions وليس CodeAssist؟
CodeAssist تطبيق ممتاز للمشاريع البسيطة، لكن مشروعنا يستخدم تقنيات حديثة
(Jetpack Compose + Kotlin 2.0 + Firebase + وحدتين) قد لا يدعمها بثبات على الهاتف.
**GitHub Actions يبني على خوادم حقيقية فيها كل الأدوات — فيعمل بنجاح مضمون.**
