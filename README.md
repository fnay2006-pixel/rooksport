# ⚽ ROOK Sport — نظام بث رياضي فاخر (تطبيقان)

نظام متكامل من **تطبيقين** بتصميم فخم (Kotlin + Jetpack Compose + Media3 + Firebase):
1. **تطبيق المستخدم** (`:app`) — للمشاهدة، تنشره للناس.
2. **تطبيق الأدمن** (`:admin`) — لوحة تحكم خاصة بك تضيف/تعدّل المباريات وروابط البث، وتظهر فوراً للمستخدمين عبر Firebase.

> 📘 **مهم:** اقرأ `دليل_Firebase.md` أولاً — يشرح ربط التطبيقين بقاعدة البيانات السحابية.

---

## ✨ أهم الميزات
- ▶️ **يشغّل روابط m3u8 (HLS)** عبر Media3 ExoPlayer — مع فرض نوع HLS تلقائياً.
- 🌐 **تحكم عن بُعد**: التطبيق يقرأ ملف `config.json` من الإنترنت عند كل فتح.
- ➕ **تضيف المباريات يدوياً** أنت بنفسك (وليس تلقائياً)، وتظهر فوراً — بدون تحديث التطبيق.
- 🔄 **تحدّث روابط البث** في أي وقت من ملف JSON واحد فقط.
- 🎨 تصميم فاخر: أسود معتدن + ذهبي + تركواز، شارات LIVE نابضة، 4 شاشات متكاملة.
- 🧭 شاشات: الرئيسية • القنوات • تفاصيل المباراة • المشغّل • حسابي (مع زر تحديث وحالة الاتصال).

---

## 🚀 كيف يعمل التحكم عن بُعد؟ (الأهم)
```
[ هاتف المستخدم ] ←—— يقرأ ——→ [ config.json على GitHub ] ←—— تعدّله أنت
```
1. ضع رابط ملف `config.json` مرة واحدة في:
   `app/src/main/java/com/rook/sport/data/RemoteConfigUrl.kt`
2. بعدها، كل إضافة مباراة أو تغيير رابط m3u8 تتم من الملف فقط.

📖 **الشرح الكامل خطوة بخطوة** في: `server/دليل_التحكم.md`
🖱️ **محرر مرئي جاهز** (يولّد JSON بدون كتابة): `server/محرر-الاعدادات.html`
📄 **نموذج جاهز**: `server/config.json`

---

## 🗂️ هيكل المشروع
```
app/src/main/java/com/rook/sport/
├── MainActivity.kt
├── data/
│   ├── RemoteConfigUrl.kt     ← ⭐ ضع رابط config.json هنا
│   ├── ContentViewModel.kt    ← يحمّل المحتوى
│   ├── remote/                ← Retrofit + نماذج JSON
│   ├── repo/                  ← ContentRepository + بيانات تجريبية احتياطية
│   └── model/                 ← نماذج التطبيق
├── ui/theme · ui/components · ui/screens
├── player/                   ← مشغّل ExoPlayer (HLS/m3u8)
└── navigation/               ← التنقل + الشريط السفلي
server/
├── config.json               ← الملف الذي تتحكم به
├── محرر-الاعدادات.html        ← محرر مرئي يولّد JSON
└── دليل_التحكم.md             ← دليل كامل بالعربي
```

## 🛠️ البناء والتشغيل
1. افتح مجلد المشروع في **Android Studio** (Hedgehog+).
2. انتظر مزامنة Gradle.
3. ضع رابط `config.json` في `RemoteConfigUrl.kt`.
4. شغّل على جهاز/محاكي (Android 7.0+ / minSdk 24).
> Android Studio يولّد `gradle-wrapper.jar` تلقائياً عند أول فتح.

---

## ⚖️ ملاحظة قانونية
- إذا لم يُضبط الرابط بعد، يعرض التطبيق بيانات تجريبية (روابط اختبار مجانية) ولا يتعطل.
- 🚫 ضع فقط روابط m3u8 **قانونية/مرخّصة أو مفتوحة المصدر**. تجنّب القنوات المشفّرة المقرصنة (مثل beIN) لأنها مخالفة وتعرّضك وحساب Google Play للحظر.
